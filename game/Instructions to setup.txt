To start programming:

Change to 64-bit project using http://stackoverflow.com/a/4364020
build glfw
add include folder to project VC++ include directories for glew/glfw
add build/src/Debug folder to VC++ library directories for glew/glfw
add .lib file name to linker/input for glew/glfw
add glew dll to exe folder

To init glfw and glew:

* Init glfw
* Create glfw window
* Init glew
